# Movie Recommender System

A research project exploring movie recommendation algorithms using the MovieLens dataset.

## Project Structure

```
├── Recommender System.ipynb                              # Main recommender system
├── Recommender System with 10M dataset.ipynb             # 10M dataset version
├── Recommender System with 10M dataset Proposed Method.ipynb  # Proposed method
├── train_ratings.csv                                     # Training data
├── test_ratings.csv                                      # Test data
├── archive/                                              # MovieLens dataset
└── ml-latest-small/                                      # MovieLens small dataset
```

## Dataset

This project uses the [MovieLens](https://grouplens.org/datasets/movielens/) dataset, which contains movie ratings collected by GroupLens Research.

## Getting Started

1. Clone this repository
2. Install dependencies:
   ```bash
   pip install pandas numpy matplotlib seaborn scikit-learn
   ```
3. Open the Jupyter notebooks to explore the analysis and models
